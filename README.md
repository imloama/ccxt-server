ccxt server
=============

ccxt rest server supports by nestjs and typeorm

## dependencies
> nodejs >= 8
> yarn

## install
```
  yarn
```

## run
```
  yarn run start
```

## run by proxy `http://127.0.0.1:1080`
```
  yarn run dev:proxy
```

## api by swagger
> http://127.0.01:3000/api
